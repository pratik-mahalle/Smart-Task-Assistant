# Documentation Assets

This folder is for documentation assets like screenshots and GIFs.

## Suggested Assets to Create

1. **demo.gif** - A screen recording of the app in action
2. **architecture.png** - A diagram of the system architecture  
3. **screenshot.png** - A static screenshot of the UI

## How to Create a Demo GIF

1. Use a screen recorder like [Kap](https://getkap.co/) (macOS) or [ScreenToGif](https://www.screentogif.com/) (Windows)
2. Record yourself using the app:
   - Add a few tasks with different priorities
   - Complete a task
   - Filter to show urgent items
   - Delete a task
3. Export as GIF (max 10MB for GitHub)
4. Place the file here as `demo.gif`

The README references `./docs/demo.gif` for the header image.
